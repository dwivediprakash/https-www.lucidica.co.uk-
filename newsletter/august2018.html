Tech Tip of the Month:

How to Increase Your Productivity on a Computer?

1.Use bigger screen: Use of a bigger screen or multiple screens (i.e. dual monitors) can help increase productivity by having more things in view at once.

2.Use an up-to-date and recent computer: Using an up-to-date computer or upgrading can help you be more efficient by reducing the load time of programs.

3.Do some clean up work: When a newer computer isn't possible, some clean up work may help your current computer run a little faster.
-Clean up your hardware
-Defragment the hard disk
-Upgrade hardware

4.Use shortcut keys: Use of shortcut keys can help you save time by keeping your hands on the keyboard instead of having to navigate a menu to select the action every time it is needed. The more an action is used, the more that using the shortcut key for it can help improve productivity.
