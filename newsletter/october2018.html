Tech Tip of the Month:

How to Launch Programs With Your Own Hotkeys

Right click the shortcut to any application in Windows

1.Head into the Properties

2.In the Shortcut tab you should see a "shortcut key" field

3.Create the Shortcut key you want by pressing a key on keyboard and hit Save

ie.: To create Ctrl + Alt + 2 , press only "2" and it will create an automatic shortcut
