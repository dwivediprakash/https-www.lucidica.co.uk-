Tech Tip of the Month:

Disable Autoplay Videos in Chrome

You can now turn off autoplay videos!

1.In the Chrome address bar, paste this: chrome://flags//

2.On your keyboard press CTRL F (or Command F on a Mac)

3.Type autoplay

4.Click the dropdown menu and set it to "document user activation required"
